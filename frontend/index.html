Project Criteria Overview

Question Types

Typ A (Pos/Neg)
Single Choice. Nur 1 richtige (Positiv) oder 1 falsche (Negativ) Antwort aus 4-5 Optionen.
Kprim
Vierfachentscheidung. Exakt 4 Optionen. Jede Aussage muss einzeln als Richtig (+) oder Falsch (-) bewertet werden.
PickS / PickN
Multiple Choice. Bei N gesuchten Antworten müssen exakt M = N × 2 Antwortoptionen existieren.

Linter Rules (Verboten)

  • Absolute Signalwörter
    "immer", "nie", "alle", "ausschließlich", "stets", "kein" geben ungewollte Lösungshinweise.
  • Doppelte Verneinungen
    Vermeiden von verwirrender Syntax (z.B. "Welche Aussage ist nicht unzutreffend?").
  • Length Cues
    Wenn eine Option auffällig länger und detaillierter ist als der Rest (meist die korrekte Antwort).
